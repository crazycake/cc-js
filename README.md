CrazyCake JS
------------

JS Core for webapps. Gulp as Task Runner, Browserify as bundler.

### Requeriments

`npm install gulp npm-release -g`

###### (Optional)

`npm install npm-check  eslint -g`

### Usage

Run `npm run watch` for watch & build.

Run `npm run build` for build only.
