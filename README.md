CrazyCake JS
------------
[![npm version](https://badge.fury.io/js/crazycake-js.svg)](https://badge.fury.io/js/crazycake-js)

JS Core for webapps. Parcel as bundler.

### Requeriments

Install

`yarn install`

###### npm tools (optional)

`npm install -g npm-check npm-release`

### Usage

watcher: `npm run watch`
